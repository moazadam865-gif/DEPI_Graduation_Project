## 📌 Project Overview
This project is a comprehensive Machine Learning pipeline and interactive dashboard designed for an E-Commerce platform. It solves 8 distinct business problems using advanced predictive modeling and data analytics to help decision-makers boost sales, retain customers, and optimize operations.

Developed as a graduation project for the **Digital Egypt Pioneers Initiative (DEPI)**.

**Author:** Moaz Adam  
**Academic Context:** Information Systems, Faculty of Computers and Artificial Intelligence, Beni Suef University.

## 🚀 Features & Business Solutions
We deployed 8 different machine learning algorithms to address critical business questions:

1. **Customer Segmentation (K-Means):** RFM analysis to cluster customers into actionable segments (VIP, Loyal, New).
2. **Late Delivery Prediction (Random Forest):** Classifies whether an order will be delivered late based on price and freight value.
3. **Sales Forecasting (XGBoost Regressor):** Time-series prediction of daily sales volume to manage inventory efficiently.
4. **Review Score Drivers (Decision Tree):** Predicts customer satisfaction (1-5 stars) and extracts feature importance (e.g., delivery days impact).
5. **Customer Churn Prediction (XGBoost Classifier):** Identifies customers at risk of leaving the platform, allowing for targeted retention campaigns.
6. **Customer Lifetime Value (Linear Regression):** Predicts the total monetary value a customer will bring to the business.
7. **Product Recommendation (KNN):** Cross-selling engine based on product similarity, price, and customer reviews.
8. **Freight Value Optimization (PCA + Multiple Regression):** Reduces physical dimensions via PCA and predicts ideal shipping costs to avoid profit loss.
